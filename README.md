# elite102-banking-app
Features: Password,Multiple Accounts, Transaction History, Account creation, Deposit, Withdrawl, and database usage.
Set up instructions: Open github, click the play button in the top right, and test the code.
Description:
This project is a banking app made with Python and SQLite. It lets you create an account, log in, and keep track of your money. You can add money, take money out, and check your balance whenever you want. It also saves all your transactions so you can see what you did before. The data stays saved even if you close the program, so your account doesn’t get lost.
